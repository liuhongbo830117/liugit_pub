Matlab codes for reproducible research

a) Description: Matlab codes for "Secrecy Capacity Analysis of Artificial Noisy MIMO Channels - an Approach Based on Ordered Eigenvalues of Wishart Matrices". This includes 9 (.m) files.

b) 12 KB in total

c) Platform: MATLAB_2016_a in macOS (You can use MATLAB_2014_b or later.)

d) Packing list:
	1) cal_mean_variances.m: This code is used to calculate the means and variances of the k-th largest eigenvalue of an uncorrelated central Whishart matrix.

	2) Lowgamma.m: Self-defined lower incomplete gamma function

	3) mycapacity.m: Numerical MIMO capacity

	4) myeigencapacity.m: Numerical capacity of the k-th eigen-subchannel

	5) myeigenpdf.m: This code is used to provide the k-th largest eigenvalue distribution (cdf and pdf) of the uncorrelated central Whishart matrix.

	6) pdf_figure.m: This code is used to provide the k-th largest eigenvalue pdf figures of uncorrelated central Whishart matrices.

	7) Theorem_3_Monte_Carlo_numerical_analysis.m: Secrecy capacity of Theorem 3

	8) Theorem_4_Monte_Carlo_numerical_analysis.m: Secrecy capacity of Theorem 4

	9) Uppergamma.m: Self-defined lower incomplete gamma function
